## Objection Quick Reference

| They Say | You Say |
|----------|---------|
| "We already have a process" | "Totally get it. Out of curiosity, how long does it take per applicant?" |
| "Too expensive" | "Fair - what are you paying now for screening?" |
| "Send me info" | "Happy to. What's the best email? I'll include a link to book 15 min if you want to see it live." |
| "Not interested" | "No problem. Mind if I ask what you're using for income verification now?" |
| "Call back later" | "Sure. What day/time works? I'll put it on my calendar." |

## Daily Checklist

Morning:
- [ ] Check Instantly for replies (respond immediately)
- [ ] Build list in Apollo (25 contacts)
- [ ] Import to HubSpot
- [ ] Add to Instantly campaign

Midday:
- [ ] Pull call list from HubSpot
- [ ] Make 20 calls
- [ ] Log each call in HubSpot
- [ ] Book any demos

End of Day:
- [ ] Update deal stages in HubSpot
- [ ] Note any learnings
- [ ] Prep tomorrow's call list
